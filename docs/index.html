EV3Dev-lang-java

A FOSS project to learn Java and create software for Mindstorms Robots using EV3Dev & the LeJOS way.

Getting Started

Lego Mindstorms ecosystem

Lego Mindstorms ecosystem is an educational way to learn the Java programming language and Robotics in an easy way. Now, it is possible to install a complete Linux distro in the third generation of the product and others companies like Mindsensors & Dexter Industries has released products which interact with Sensors & Actuators from Lego Mindstorms ecosystem and that boards can use the Power of the popular board Raspberry Pi 3.

Features included in the whole project

Lego Mindstorms features

  • Regulated Motor Support
  • Unregulated Motor Support
  • Sensor Support (EV3 sensors)
  • Sounds Support
  • EV3 LCD Support

Robotics

  • OpenCV support
  • eSpeak TTS (Text to speech) support
  • LeJOS Sensor filter support
  • LeJOS local navigation stack support
  • RPLidar A1/A2 (2D Lidar) support

Platforms

  • EV3 Brick
  • BrickPi+
  • BrickPi 3
  • PiStorms

Java features

  • OpenJDK Java 10 support
  • Java profiling tools support
  • Logging support based on SLF4J
  • Support for ELK stack

OpenJDK 10 supported Learn Java with the latest version

Debian Jessie/Stretch Learn Linux in an easy way

Getting Started

The project provides a Template project to help you in the process to test the project in an easy way.

1. Install EV3Dev in your Brick

To use this project, you need to install EV3Dev in your Brick and establish a Internet connection. EV3Dev project provide a easy way to install Debian Linux on your Brick.

Getting Started with EV3Dev

2. Clone the project template-project-gradle

Clone the project and edit the file config.gradle to set the IP of your brick. The project provides some Gradle tasks to install Java and execute your programs in an easy way.

Download template-project-gradle 

        
$ git clone https://github.com/ev3dev-lang-java/template_project_gradle
$ cd template_project_gradle
$ vi config.gradle
        
      
        
remotes {
    ev3dev {
        host = '192.168.1.86'
        user = 'robot'
        password = 'maker'
    }
}
        
      
        
$ ./gradlew testConnection
$ ./gradlew getInstaller
$ ./gradlew installJava
$ ./gradlew clean deployAndRun
